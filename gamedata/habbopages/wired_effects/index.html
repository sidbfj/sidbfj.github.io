Efeitos Wired

Efeitos Wired

Como funciona o Efeito?

EFEITO WIRED: Mudar estado do Mobi

Este Efeito muda o estado do Mobi escolhido como se alguém tivesse clicando duas vezes nele

EFEITO WIRED: Teleportar para Mobi

O efeito teleporta um Habbo para um Mobi. Se selecionar mais de um Mobi, o Habbo é teleportado aleatoriamente para um deles

EFEITO WIRED: Teletransportar Bot ao Mobi

Esse efeito teletransporta o Bot selecionado a um dos Mobis escolhidos

EFEITO WIRED: Bot muda de roupa

Esse efeito muda a roupa do Bot selecionado

EFEITO WIRED: Bot vá ao Mobi

Esse efeito faz com que o Bot selecionado se mova a um dos Mobis escolhidos

EFEITO WIRED: Bot fala ou sussurra a todos os usuários

Esse efeito faz com que o Bot selecionado fale ou grite a todos os usuários

EFEITO WIRED: Bot fala ou sussurra

Esse efeito faz com que o Bot fale ou sussurre ao usuário ativado

EFEITO WIRED: Bot seg

Este efeito faz com que o Bot siga o usuário ativado

EFEITO WIRED: Bot dá item de mão

Este efeito faz com que o Bot dê o item de mão selecionado

EFEITO WIRED: Kickar Usuário

Esse efeito chuta um usuário dependendo dos critérios de acionamento

EFEITO WIRED: Mostrar mensagem

Este Efeito mostra uma mensagem secreta apenas para os Habbos que o ativaram

EFEITO WIRED: Mover e Rotacionar Mobi

O efeito move e rotaciona o Mobi selecionado

EFEITO WIRED: Mobis que voltam à programação original

Este Efeito faz o Mobi selecionado voltar ao estado original que você o programou

EFEITO WIRED: Faz pilhas de wireds

Esse Wired permite selecionar 5 pilhas de wireds para executar de um mesmo disparador, não importando as condições atuais ou disparadores

EFEITO WIRED: Reiniciar tempo

Este efeito reinicia Ativadores que usam temporizadores

EFEITO WIRED: Dar Pontos

Este Efeito dá uma quantidade x de pontos para os jogadores. Você poderá criar grandes jogos!

EXTRA WIRED: Efeito Aleatório

Apenas um Efeito aleatório poderá ser feito por vez

EXTRA WIRED: Efeito Inédito

Apenas um Efeito inédito pode ser ativado por vez

EFEITO WIRED: Perseguir

Este efeito move o mobi para o usuário mais próximo dele na mesma linha ou coluna. Se tiver um usuário próximo a este mobi, o DISPARADOR WIRED: Batida irá ativar

EFEITO WIRED: Fugir

Quando o Mobi disparado vai uma Telha mais longe do avatar mais próximo

EFEITO WIRED: Pontuação recebida

Dá pontuação para a Equipe pré-definida

EFEITO WIRED: Entrar na Equipe

Disparar o avatar entrará na equipe pré-definida

EFEITO WIRED: Sair da Equipe

Disparar o avatar abandonará a equipe pré-definida

EFEITO WIRED: Mudar Direção Mobi

Esse Efeito muda a direção do Mobi. Quando o movimento é bloqueado, o Mobi terá sua direção trocada segundo sua configuração. Se há um usuário perto do Mobi, o Wired Ativador será ativado
