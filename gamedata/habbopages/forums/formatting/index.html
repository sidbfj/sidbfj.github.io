h1. Formatação dos Fórums

Formatação básica

Use *bold* para texto em negrito, _italic_ - para texto emitálico.

Comece uma linha com o símbolo > para fazer uma citação, como:
> esta é uma citação

Refererência

Para citar outro Habbo, simplesmente use o formato '@Habboname'. Isso se transforma num hyperlink direcionado ao perfil do Habbo, e o usuário receberá uma notificação (ainda em fase de teste).

Resposta

Você pode responder mensagens individuais clicando no botão verde "Responder" da barra de mensagens. Também pode responder várias mensagens ao mesmo tempo, deixando a janela "Escrever Mensagem" aberta, clicando em mais botões "Responder" na página principal do Fórum.
