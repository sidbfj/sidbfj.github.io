Condiçoes Wired

Como funciona o condição?

CONDIÇÃO WIRED: Membro do Grupo

Condição ocorre quando é um quarto de grupo ou o usuário é membro daquele grupo

CONDIÇÃO WIRED: Ter Mobi em cima

O efeito funcionará se um ou mais dos Mobis selecionados tiverem Mobi em cima

CONDIÇÃO WIRED: Usuário tem item de mão

Ativador funciona quando o usuário está segurando o item de mão

CONDIÇÃO WIRED NEGATIVA: NÃO há Mobis em cima

Ativador é permitido se NENHUM dos Mobis escolhidos tiver outro Mobi em cima

CONDIÇÃO WIRED NEGATIVA: Mobis NÃO têm usuários em cima

Ativador é permitido se NENHUM dos Mobis escolhidos tiver usuários em cima dele

CONDIÇÃO WIRED NEGATIVA: NÃO é membro de grupo

Condição ocorre quando é um quarto de grupo e o usuário NÃO é um membro do grupo

CONDIÇÃO WIRED NEGATIVA: NÃO é membro do time

Condição aceita quando usuário NÃO é membro de um time pré-determinado

CONDIÇÃO WIRED NEGATIVA: O estado do Mobi NÃO coincide

Ativador não é permitido se o estado dos Mobis escolhidos NÃO coincidir

CONDIÇÃO WIRED: Tipo de Mobi NÃO coincide

Ativador é permitido se um Mobi ativador NÃO é um dos Mobis escolhidos e NÃO é do mesmo tipo

CONDIÇÃO WIRED NEGATIVA: Ativador NÃO está no Mobi

Ativador é permitido se quem o ativa NÃO está em cima de um dos Mobis escolhidos

CONDIÇÃO WIRED: NÃO há usuários suficientes no quarto

Ativador não é permitido se o número de usuários em um quarto NÃO equivale aos limites pré-determinados

CONDIÇÃO WIRED: Membro da Equipe

A condição passa quando o avatar é um membro da equipe pré-definida

CONDIÇÃO WIRED: Mobis com Habbos

Ativador funcionará se todos os Mobis selecionados tiverem Habbos em cima

CONDIÇÃO WIRED: Estado e Posição do Mobi combinam

Ativador funcionará se a posição e o estado do Mobi combinarem

CONDIÇÃO WIRED: Tipo de Mobi coincide

Ativador é permitido se o Mobi ativador é um dos escolhidos ou é do mesmo tipo

CONDIÇÃO WIRED: Menos de X segundos passados

Ativadores funcionarão se menos de x segundos tiverem passado desde o reinicio do temporizador

CONDIÇÃO WIRED: Mais de X segundos passados

Ativadores funcionarão se mais de x segundos tiverem passado desde o reinicio do temporizador

CONDIÇÃO WIRED: Habbo Ativador está no Mobi

Ativação é permitida se o Habbo Ativador estiver no Mobi selecionado

CONDIÇÃO WIRED: Número de usuários no quarto

Ativador é permitido se o número de usuários no quarto estiver dentro dos limites definidos
